# Q-Token Authentication System

## 📌 Description
The Q-Token System is a secure authentication solution that provides an extra layer of protection using multi-factor authentication (2FA). It generates one-time passwords (OTP) for secure user login and access control.

## 🚀 Features
- Two-Factor Authentication (2FA)
- One-Time Password (OTP) generation
- Secure login system
- Biometric authentication support (if implemented)
- Works without continuous internet connection
